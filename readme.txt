RegWiz is a Event Registration App for Techtatva 2010, MIT's National Level Tech Fest.

It is based on PHP and uses OOPS.

1. Download the zip.
2. Extract it to your www/htdocs folder on your webserver. Eg. /srv/www/regwiz
3. Create a database and replace the database name, DB username, DB password in class.inc.php // DBNAME
4. Import the "techtatva.sql" file into the database
5. Use the browser on localhost to check the RegWiz web application. http://localhost/regwiz

Make your college event easy :)
